# Recorded synthetic FMP fixtures

Every body in this directory is original synthetic data created for PEAS. No response body,
headline, release prose, image, URL, credential, or header was copied from FMP. The dialect is the
closed `peas-fmp-press-release-synthetic-v1` contract and is not represented as a live FMP schema.

`latest.json`, `search.json`, `duplicate.json`, `correction.json`, and `missing-time.json` are valid
UTF-8 collections. `malformed.json` is intentionally invalid. Exact and one-over byte/item/token/
depth/string bounds are generated in the contract test and are not checked in as large files.
